# Software Security Project Backend

Backend API for the Software Security Project.

## Overview

This backend provides:

- User authentication
- JWT-based protected routes
- Task-related APIs
- File-related APIs
- User and admin route groups

## Tech Stack

- Node.js
- Express.js
- MySQL
- bcrypt
- JSON Web Token (JWT)
- helmet
- express-rate-limit
- morgan
